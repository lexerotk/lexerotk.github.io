Your Privacy

Some explanation for paranoids.
Some people thinks this website is an IP grabber. But in fact, no. I do not store any IP addresses. IP addresses are only being collected to check the security of connection. Your IP address is only stored in your own browser's local storage for 3 hours as trusted IP address to not overdo the security check. The algorithm of the security check is; the website checks your browser's local storage to see if you have the trusted key. If you don't have that or your IP address is changed recently or trusted key is older than 3 hours system collects your IP address with an API and then sends this IP address to a different API to see if your IP address is suspicious or not. Then by the result system writes the trusted key to local storage or sends you to "blocked" page. While these happens IP addreses not stored in anyway, just sent to an API. My website is open-source, meaning you can inspect the code.
